# 写一个整体的网页
这次的任务感觉即难又简单，难在发现还有很多盲区，css掌握的还是不够牢靠，简单在发现页面复杂自己还是可以很好的实现
总结一下：
## 设置背景图片的尺寸
background-size
包括两个值：width height
以及图片的覆盖方式
cover：完全覆盖，图片可能不完全显示
contain：锁定纵横比最大的图片，保证完全显示

## 垂直居中
vertial-align：middle;

## 伪类
hover：鼠标移入
active：点击
visited:已经访问过
